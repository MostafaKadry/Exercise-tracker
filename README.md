# Exercise Tracker

## Overview

The Exercise Tracker is a full-stack MERN (MongoDB, Express.js, React.js, Node.js) application that helps users keep track of their daily exercise routines. The application allows users to create, read, update, and delete exercises, providing a comprehensive way to manage their fitness activities.

## Features

- **User Authentication**: Secure login and registration system.
- **CRUD Operations**: Create, read, update, and delete exercises.
- **Exercise Categories**: Various exercise categories to choose from.
- **Responsive Design**: Mobile-friendly and responsive layout using Bootstrap.
- **Interactive UI**: User-friendly interface with React.js.

## Technologies Used

- **Frontend**:
  - React.js
  - Bootstrap
  - CSS3
  - HTML5

- **Backend**:
  - Node.js
  - Express.js
  - MongoDB
  - Mongoose

## Installation

1. **Clone the repository**:
    ```sh
    git clone https://github.com/MostafaKadry/Exercise-tracker.git
    cd Exercise-tracker
    ```

2. **Install backend dependencies**:
    ```sh
    cd backend
    npm install
    ```

3. **Install frontend dependencies**:
    ```sh
    cd ../frontend
    npm install
    ```

4. **Run the application**:
    - **Backend**:
        ```sh
        cd backend
        npm start
        ```
    - **Frontend**:
        ```sh
        cd ../frontend
        npm start
        ```

5. **Access the application**:
    Open your web browser and go to `http://localhost:3000`.

## Usage

1. **Register**: Create a new account.
2. **Login**: Access your account.
3. **Add Exercise**: Log your exercises with details such as duration, type, and date.
4. **View Exercises**: See a list of all your logged exercises.
5. **Edit Exercise**: Update details of an existing exercise.
6. **Delete Exercise**: Remove an exercise from the list.

## Project Structure

- `backend/`: Contains the backend code using Node.js and Express.js.
  - `models/`: Mongoose schemas and models.
  - `routes/`: API routes.
  - `controllers/`: Route controllers.
  - `config/`: Configuration files (e.g., database connection).
  - `server.js`: Entry point for the backend server.

- `frontend/`: Contains the frontend code using React.js.
  - `src/components/`: React components.
  - `src/pages/`: Different pages of the application.
  - `src/App.js`: Main application component.
  - `src/index.js`: Entry point for the frontend.
